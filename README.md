# EduNotify: Private School-to-Parent Voice Alerts 🏫🎓

EduNotify is a specialized communication tool designed for **Private K-12 Schools** and educational institutions. It facilitates automated, critical, and time-sensitive alerts to parents using the Twilio Programmable Voice API.

## 🌟 Use Case Scenario
Private schools require high-priority communication channels. In regions where mobile apps or email notifications are often ignored or delayed due to unstable internet, a direct phone call is the most effective and reliable method.

EduNotify automates notifications for:
- **Student Absence:** Immediate notification if a child is marked absent without prior notice.
- **Emergency Closures:** Fast broadcast of closures due to weather or other emergencies.
- **Parent-Teacher Meetings:** Reminders for scheduled academic review sessions.
- **Tuition/Administrative Reminders:** Gentle reminders for pending administrative tasks or tuition deadlines.

## 🛠 Tech Stack
- **Backend:** Python / Flask
- **Voice Engine:** Twilio Voice API (TwiML / Text-to-Speech)
- **Data Integration:** (MVP) Reads standardized student roster data.

## 🔒 Compliance & Privacy (Sole Use)
- **School-to-Parent Only:** This is a private utility for registered school parents. It does not support bulk marketing or unsolicited communications.
- **Opt-in Required:** Call lists are generated solely from parent contact information provided and verified at the school's administrative office.
- **No Telemarketing:** The system strictly delivers informational and operational messages.
- **User Control:** Parents can request to pause or unsubscribe from automated calls via the school's admin office.

This project aims to enhance student safety and parent engagement in the modern private educational environment.
